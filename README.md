### Try it yourself: 
[Memory Box](https://ayanisation.github.io/Memory_box/)

## Setup:

1. cd client
2. npm install
3. npm start

![Feed](https://user-images.githubusercontent.com/48587903/111036345-cc96a780-8444-11eb-9890-f46ff6a5b01d.png)

- Create new posts
- Update the existing posts of yours
